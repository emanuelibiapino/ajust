Ajuda

1º Passo: Ao acessar à página inicial do site, selecione a cidade que deseja realizar a denúncia.

2º Passo: Após selecionar a cidade. Clique no botão Iniciar!

Passo 3º: Ao selecionar uma categoria, você verá todos os problemas em aberto.
Caso queira realizar uma nova denúnca, clique no botão: Relatar novo problema!

4º Passo: Preencha todos os campos com as informações pedidas.

5º Passo: Após preencher todos os valores. Clique no botão Informar o problema para enviar a reclamação.
